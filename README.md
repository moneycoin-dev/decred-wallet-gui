# decred-wallet
Decred Wallet UI.

###### Features
* Multithreaded
* Cached results
..* Realtime updates on actions
* Accounts
..* View all
..* Add accounts
..* Rename accounts
* Transactions
..* List Transactions
* Balance
..* View total by account
..* View spendable by account
..* View locked by account
* Send coins
* Receive coins address for each account
..* Get new address
* Logs
..* Daemon log
..* Wallet log
..* Gui log
* Language
..* Supports multiple languge though conf files
